# Sigma Project Ninja

Sigma Project Ninja is a data analysis project focused on building business metrics and extracting actionable insights from real-world datasets.

## Scope
The project involves:
- Data cleaning and transformation
- KPI definition and tracking
- Querying large datasets (BigQuery)
- Building dashboards for performance analysis

## Tools
- SQL (BigQuery)
- Python
- Looker Studio

## Example
A typical use case consists of analyzing time-series data to monitor business performance. For instance, tracking the evolution of key indicators such as total activity, growth rate, or risk exposure over a defined period.

## Use Case
This project is particularly useful in contexts where decision-making relies on quantitative analysis, such as financial performance monitoring, sales tracking, or risk assessment.
